## Setup
 
### Prerequisites
 
- Install Tesseract OCR: https://github.com/tesseract-ocr/tesseract
- Install Ollama: https://ollama.com
 
Then pull the vision model:
```bash
ollama pull llava
```
 
### Install Python Dependencies
 
```bash
pip install -r requirements.txt
```
